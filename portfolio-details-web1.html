Secure FundRaiser Chain (Fundereum)

A decentralized fundraising platform built on Ethereum blockchain with smart contracts, enabling transparent cryptocurrency donations and empowering donors with comprehensive transaction reports.

Project Information

  • Category: Blockchain & Web Development
  • Role: Full-Stack Developer
  • Year: 2023-2024
  • Technologies: React, MongoDB, Node.js, Solidity, MetaMask, Ethereum
  • Type: Decentralized Application (DApp)

Project Description

Objective: Designed and developed a secure, transparent crowdfunding platform leveraging blockchain technology to enable NGOs and organizations to raise maximum funds through cryptocurrency donations, with built-in incentives and complete transaction transparency.

Technical Architecture: Built a full-stack decentralized application (DApp) using React for the modern, responsive frontend interface. The platform features an intuitive dashboard where users can view all campaigns, filter by "All Campaigns" or "Your Campaigns", and create new fundraising initiatives. Integrated Node.js for backend services and MongoDB for efficient data management of campaign information and user data.

Blockchain Implementation: Developed and deployed smart contracts on the Ethereum network to manage fundraising campaigns, process donations, and maintain immutable transaction records. Integrated MetaMask wallet seamlessly, displaying connected account addresses and ETH balances directly in the interface. Users can interact with campaigns, make donations, and track their contributions in real-time.

User Interface Features: The platform includes a comprehensive campaign management system where users can create campaigns with custom images, titles, and descriptions. Each campaign card displays status indicators (active/inactive), campaign details, and donation information. The interface supports account management with SignUp and Account management features.

Key Features: Secure cryptocurrency donations via MetaMask integration, transparent transaction records on blockchain, real-time balance tracking, campaign creation and management, comprehensive donor reports with detailed analytics, incentive mechanisms to maximize fundraising, and intuitive filtering system for campaign discovery.

Impact: This project demonstrates expertise in blockchain development, smart contract programming, and full-stack web development. The platform empowers NGOs and individuals with a secure, transparent way to raise funds while providing donors with complete visibility into campaign progress and transaction history. The user-friendly interface makes blockchain technology accessible to non-technical users.

Skills Demonstrated: Blockchain Development, Solidity, Smart Contracts, Ethereum, DApp Development, React, Node.js, MongoDB, MetaMask Integration, Web3.js, Full-Stack Development, Cryptocurrency Transactions, UI/UX Design, Real-time Data Display
